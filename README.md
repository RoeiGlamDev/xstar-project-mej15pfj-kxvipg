# HouseIL Documentation

Welcome to the HouseIL project documentation. HouseIL is a luxury Airbnb website designed to offer an elegant and high-end experience for tech-savvy travelers seeking unique accommodations. This README provides essential information to set up and navigate the HouseIL platform.

## Project Overview

HouseIL is built with a modern dark design aesthetic, leveraging a color palette of pink and black to create a visually striking interface. Our mission is to redefine the way high-end travelers book their stays, showcasing properties that merge luxury with cutting-edge technology.

## Features

- Luxurious dark-themed interface
- Custom property listings with high-resolution images
- User-friendly navigation for an exceptional booking experience
- Responsive design, ensuring usability on all devices
- Integrated payment system for seamless transactions
- User profile management for personalized experiences

## Tech Stack

- Frontend: React.js
- Backend: Node.js with Express
- Database: MongoDB
- Styling: CSS3 with Tailwind CSS for rapid customization

## Setup Guide

### Prerequisites

Ensure you have the following installed on your machine:

- Node.js (v14 or later)
- npm (Node Package Manager)

### Installation Steps

1. Clone the Repository

   Open your terminal and clone the HouseIL repository:
   git clone https://github.com/YourUsername/HouseIL.git
   cd HouseIL
   2. Install Dependencies

   Install the required packages by running:
   npm install
   3. Set Up Environment Variables

   Create a .env file in the root directory and configure your environment variables according to the following template:
   DATABASE_URL=your_mongodb_connection_string
   SECRET_KEY=your_secret_key
   4. Run the Application

   Start the development server using:
   npm start
   Navigate to http://localhost:3000 in your web browser to view the HouseIL application.
